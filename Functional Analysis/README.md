# Functional Analysis

Hand-calculation workflow used to check the ANSYS static structural result.

## Files

- [calculation_notes.md](calculation_notes.md): GitHub-readable formula notes
- [Calculations/rectangular_beam_calculations.pdf](Calculations/rectangular_beam_calculations.pdf): polished calculation PDF
- [Calculations/rectangular_beam_handwritten_calculations.pdf](Calculations/rectangular_beam_handwritten_calculations.pdf): handwritten calculation scan
- [Calculations/rectangular_beam_calculations.tex](Calculations/rectangular_beam_calculations.tex): LaTeX source for the calculation PDF

The calculation builds from geometry and units through area, surface traction, bending moment, section inertia, bending stress, tip deflection, von Mises stress, and ANSYS comparison.
